# 项目简介

## 核心功能
- 基于YOLOv8的水面漂浮物检测系统，支持11类漂浮物识别
- 提供完整源码、3000张标注数据集、训练好的模型及训练结果
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 提供UI界面，实时显示目标位置、目标总数、置信度等信息
- 支持自定义检测参数：置信度阈值、IOU阈值
- 提供多种显示选项：标签显示、原图显示、检测模型选择
- 支持批量检测功能，可查看所有检测结果
- 提供完整的运行教程和环境配置说明

## 适用场景
- 水面环境监测与管理
- 环保部门的水面漂浮物普查
- 水库、湖泊等水域的污染监测
- 水面垃圾自动识别与分类
- 科研机构的水面漂浮物研究

## 技术特点
- 技术栈：Python 3.9 + OpenCV + PyQt5 + Ultralytics + PyTorch 1.9
- 开发环境：PyCharm + Anaconda 或 VSCode + Anaconda
- 预训练模型可直接使用，环境配置成功即可运行
- 提供完整的UI界面源文件和图标资源
- 模块化设计，便于功能扩展和二次开发
